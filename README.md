# ポラーノワークス 見学案内サイト

架空の就労継続支援B型事業所「ポラーノワークス」の見学案内サイトです。

## 制作目的

見学前に不安を感じている本人が、利用を急いで決めなくてもよいことを理解し、自分のペースで見学について相談できるようにするための1ページサイトです。

## 主な工夫

- 本人向けの安心導線と、ご家族・支援機関向けの客観情報を分けて配置
- 「予約」ではなく「見学について相談する」入口としてフォームを設計
- 必須項目を `お名前` と `連絡先` の2つに限定
- 見学の流れを1列の縦型タイムラインで整理
- 写真で見る道順セクションは、装飾より実用的なカード設計を優先
- スマホで読みやすく、押しやすい余白と文字サイズに調整
